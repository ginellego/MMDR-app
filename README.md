# MMDR-app
Mobile app for Manitoba Mutts Dog Rescue team for communication, support and collaboration among the organization volunteers. 


# Running Instructions

Have an active cluster on MongoDB. MMDR-app

node -v
npm -v

Run the backend

cd backend
npm init -y
npm install express

cd /Users/ginelletemull/Documents-copy/Manitoba Mutts/MMDR-app

Install dependencies

npm install --save-dev babel-plugin-module-resolver


Install Expo (one-time only)
npm install -g expo
expo --version

Run Expo 
inpx expo start (-c --> clear cache)
